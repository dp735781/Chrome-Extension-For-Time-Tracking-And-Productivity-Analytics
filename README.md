*DISCRIPTION*:
A powerful and lightweight Chrome extension designed to help users track their browsing time and analyze productivity across different websites. This tool provides real-time insights into how time is spent online, enabling better focus, improved habits, and smarter time management.

🚀 Features

⏱️ Real-Time Time Tracking
Automatically tracks the time spent on each website.

📊 Productivity Analytics Dashboard
Visualizes usage data with easy-to-understand insights.

🌐 Website Usage Monitoring
Categorizes websites to identify productive vs. non-productive activity.

🔔 Idle Detection
Detects inactive time to ensure accurate tracking.

📅 Daily & Weekly Reports
Provides summaries of browsing behavior over time.

⚡ Lightweight & Fast
Minimal performance impact on your browser.

🛠️ Tech Stack

JavaScript (Vanilla JS)

Chrome Extension APIs

HTML & CSS

📂 How It Works

The extension uses Chrome APIs like tabs, storage, and idle to monitor active tabs and record time spent. Data is stored locally and displayed through a simple, user-friendly interface for analysis.

Students tracking study vs. distraction time

Developers monitoring productivity

Professionals improving work efficiency
